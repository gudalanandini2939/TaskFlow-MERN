# 🚀 TaskFlow MERN

A Full Stack Task Management Dashboard built using MongoDB, Express.js, React.js, and Node.js.

## Features

* User Authentication (JWT)
* Add, Edit, Delete Tasks
* Complete / Pending Tasks
* Search & Filters
* Priority Levels
* Due Dates
* Productivity Tracking
* Dark & Light Theme
* Responsive Dashboard UI

## Tech Stack

* MongoDB
* Express.js
* React.js
* Node.js
* JWT Authentication
